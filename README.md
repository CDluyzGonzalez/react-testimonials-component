# React Testimonials Component 🗣️

Componente de testimonios desarrollado con React y Vite, diseñado para mostrar opiniones de estudiantes de manera visual y atractiva.

## 🚀 Tecnologías utilizadas

- React 18
- Vite
- CSS3

## ✨ Características

- Visualización de testimonios con foto, nombre, país, cargo y empresa
- Diseño responsive
- Componente reutilizable

## ⚙️ Instalación y ejecución

1. Clonar el repositorio:
```bash
git clone https://github.com/CDluyzGonzalez/react-testimonials-component.git
cd react-testimonials-component
```

2. Instalar dependencias: 
```bash
npm install
```

3. Ejecutar el servidor: 
```bash
node app.js
```
Servidor disponible en: http://localhost:5173
